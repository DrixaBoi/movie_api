This API will give the user access to information about movies, directors, and genres so they can learn more about movies they've watched or are interested in.

URL Endpoints

User Request URL HTTP Method Response
View a list of movies /movies GET A JSON object holding data about movies
View data about a single movie /movies/:title GET A JSON object holding data about a particular movie
Example:
{ id: "", title: "", description: "", genre: "", director: "", actors: "", rating: "", release: "", imagePath: "", Featured: "" }
Add/Post a new movie /movies POST Add a JSON holding data on the added movie
Example:
{ "Title": "", "Description": "", "Genre": "", "Director": "", "Actors": "", "Rating": "", "Release": "", "ImagePath": "", "Featured": "" }
View a list of genres /genres GET A JSON holding data about genres
View data about a specific genre /genres/:name GET A JSON object holding data about a genre and movies of that genre
Example:
{ name: "", description: "" }
View a specific genre /genres/:genre GET A JSON holding data about genres
View data about a specific genre /genres/:name GET A JSON object holding data about a genre and movies of that genre
Example:
{ name: "", description: "" }
View list of all directors /directors GET A JSON object holding data about all directors
Example:
{ id: "", name: "", bio: "", Movies: "", birth: "", death: "" }
View data about a director by name /director/:directorName GET A JSON object holding data about a director
Example:
{ id: "", name: "", bio: "", Movies: "", birth: "", death: "" }
View a list of Users /users GET A JSON object with data about all users
Example:
{ id: "", username: "", password: "", email: "", favorites: "" }
View a specific user /users/:Username GET A JSON object with data about a user
Example:
{ id: "", username: "", password: "", email: "", favorites: "" }
Registering a new user /users POST Create JSON object with data about a new user
Example:
{ id: "", username: "", password: "", email: "", favorites: "" }
Allow users to update their personal info /users/:Username PUT Update JSON object with new datalist
Example:
{ id: "", username: "", password: "", email: "", favorites: "" }
Allow users to add movies to their favorites list /users/:Username/movies/:MovieID PUT Adds a object to the users favorites data Example confirmation: "Movie was successfully added to favorites!"
Allow users to delete movies from their favorites list /users/:Username/movies/:MovieID DELETE Deletes a object from the users favorites data Example confirmation: "Movie was successfully removed from favorites."
Allow users to delete their registered account /users/:Username DELETE Removes the users data from the API Example confirmation: "Your account was successfully removed."
